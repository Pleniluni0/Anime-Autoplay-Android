Política de Privacidad de Anime AutoPlay

Esta extensión ha sido diseñada respetando la privacidad del usuario por encima de todo.

Recolección de Datos: No recolectamos, almacenamos ni transmitimos ningún tipo de dato personal, historial de navegación o información técnica del dispositivo a servidores externos.

Uso de Almacenamiento: Utilizamos el API browser.storage.sync exclusivamente para guardar las preferencias de configuración del usuario (tiempos de intro, activación de autoplay, etc.). Estos datos se sincronizan únicamente a través de la cuenta de Firefox del usuario y no son accesibles por nosotros.

Comunicaciones: La extensión no contiene telemetría ni se comunica con ningún servidor de terceros, excepto para cargar las fuentes de Google Fonts en el menú de configuración (popup).

Código Abierto: El código es transparente y está disponible para auditoría en nuestra página oficial de GitHub.

Al utilizar esta extensión, aceptas que tus datos de configuración se guarden de forma local o sincronizada en tu navegador.
